GumptionChain Passkey Wallet Demo

Real WebAuthn-PRF + IndexedDB adapters. Must be served over a secure context (https:// or http://localhost) in a PRF-capable browser.

Backup / recovery

Acts on the wallet from the most recent Enroll or Unlock above (currentWallet).

Sign message (#2.4)

Signs arbitrary text into a portable gc-msg-v1 proof using currentWallet. Verifies off-chain in JS and Python.

Verify message (#2.4)

Paste either the JSON proof or the armored block; verification is self-contained (no wallet required).

Output

(idle)

Inspect the stored record in devtools → Application → IndexedDB → gc-wallet. It should hold only {version, address, credentialId, iv, ciphertext} — no plaintext key.
