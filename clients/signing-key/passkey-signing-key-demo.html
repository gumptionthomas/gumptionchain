GumptionChain Passkey SigningKey Demo

Real WebAuthn-PRF + IndexedDB adapters. Must be served over a secure context (https:// or http://localhost) in a PRF-capable browser.

Backup / recovery

Acts on the signing_key from the most recent Enroll or Unlock above (currentSigningKey).

Sign message (#2.4)

Signs arbitrary text into a portable gc-msg-v1 proof using currentSigningKey. Verifies off-chain in JS and Python.

Verify message (#2.4)

Paste either the JSON proof or the armored block; verification is self-contained (no signing_key required).

Stake attestation (#176b)

Builds a "Verified on GumptionChain" stake claim, signs it with currentSigningKey into a gc-msg-v1 proof, then verifies it against on-chain provenance fetched from this node. Verification needs reader access — a public node sets READER_ADDRESSES=["*"].
