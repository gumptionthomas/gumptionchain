{# Net stance for a subject: which side leads (support vs opposition) and by how much. Magnitude + word so the meaning is unambiguous (no implicit sign convention). Green = net supported, red = net opposed, muted = even. #} {% macro net_stance(opposition, support) -%} {%- set net = support - opposition -%} {%- if net > 0 %}{{ net }} supported {%- elif net < 0 %}{{ -net }} opposed {%- else %}even{% endif -%} {%- endmacro %}
